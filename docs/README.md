# lotto 기능 구현

## 기능목록

- [] 로또 구입 금액 입력 받기
- [] 로또 번호 입력 받기
  - [] 1~45 숫자 범위로 중복되지 않는 6개 숫자와 보너스 숫자 1개 입력
  - [] 잘못된 값 입력시 에러 출력
- [] 발행한 번호 오름차순으로 출력하기
- [] 로또 숫자 랜덤 받기
  - [] 1~45 숫자 범위로 중복 없이 받기
- [] 번호를 비교해서 일치 개수 출력
  - [] 1개부터 6개 일치까지 비교
    - [] 일치 개수 별로 당첨금액 출력
  - [] 구입 금액과 당첨 금액의 비율 측정
  - [] 수익률 소수점 2자리에서 반올림하여 출력
- [] 기능 목록 단위로 테스트 구현
